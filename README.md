# 📄 Resume-Parser

## To start the FastAPI application, use the following command:

```bash
uvicorn main:app --reload
```

A lightweight, AI-powered resume parser built using Python and Azure-OpenAI tools. This application allows users to upload resumes (PDF format) and extract structured information in JSON format for easier review and processing.

### Note: All outputs will get written into "__DATA__" folder

---

## 🚀 Overview

This app is built for job seekers and recruiters to test whether resumes are **ATS (Applicant Tracking System) friendly**. It uses natural language processing (NLP) to extract key details and present them in a structured way. If the app can parse your details correctly, your resume is likely readable by ATS software.

---

## ✨ Features

- Extracts:
  - Full Name
  - Email, Phone Number
  - Skills
  - Education
  - Work Experience
  - Projects & Certifications
  - Languages
- JSON output for easy integration
- Supports PDF resumes
- Web interface with file upload 
- Easy-to-modify and extend
